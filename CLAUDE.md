# Skagway Tours — Claude Instructions

## Content Writing

Before writing any website copy, tour descriptions, blog posts, landing pages, or FareHarbor listings, read the following docs:

- `/docs/master-content-prompt.md` — the core prompt framework for all conversion-focused copy
- `/docs/content-playbook.md` — full copywriting playbook with principles, personas, frameworks, and tour-specific guidance

Apply these guidelines to every piece of copy written for this project. No exceptions.

## Code & Development

- Color palette: `#292546` (ink), `#DD5154` (ember), `#F6F1E7` (paper). No other colors unless explicitly approved.
- Eyebrow text: Caveat 700, 1.75rem, no forced uppercase.
- Page hero gradients: blue (`#1f1c39 → #292546 → #3d3870`), never green.
- Container max-width: 1300px.
- CSS class prefixes: page-scoped (e.g. `.emp-*`, `.psp-*`, `.cnl-*`).
- All changes go to feature branch `claude/...` unless user says otherwise.
