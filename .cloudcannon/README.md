# Skagway Tours — editor guide

Welcome. This site's content lives in CloudCannon. Here's what each section of
the sidebar does.

## Content

- **[Tours](cloudcannon:collections/tours)** — one entry per tour (White Pass &
  Yukon Combo, Wild Adventure Yukon, etc.). Edit title, summary, price,
  duration, highlights, featured image, and the long description. Use **+ Add**
  to create a new tour; new tours start as drafts.
- **[Blog](cloudcannon:collections/posts)** — trip reports, field notes, and
  news. Use **+ Add** to start a new post. New posts start as drafts so you can
  work on them without publishing.
- **[Pages](cloudcannon:collections/pages)** — top-level pages like About,
  Employment, Passport Requirements. Most one-off pages are here.
- **[FAQs](cloudcannon:collections/faqs)** — individual questions and answers.
  They're grouped by category on the public FAQs page, in the order set by the
  `order` field (lower numbers appear first).

## Configuration

- **[Site settings](cloudcannon:collections/site)** — site name, tagline,
  phone, email, FareHarbor booking config, the homepage hero video and promo
  banner, social links, the site navigation menu, and the footer columns.
  Change these here and they update everywhere on the site.

## Editing tips

- **Visual editor** opens by default on tours, blog posts, and pages. Click on
  any highlighted element in the preview to edit it inline.
- **Content editor** is best when writing or editing the long body text of a
  page, tour, or post.
- **Data editor** lets you edit every field at once, including things that
  don't show up inline (like draft status or sort order).
- **New tour / post preview** — brand-new tours and posts aren't built until
  saved, so the preview shows an existing page until then. That's expected.
- **Drafts** — posts and tours have a **Draft** switch. Draft content is
  hidden from the live site until you turn the switch off.

If something isn't editable that you think should be, let the dev team know —
we can wire it up.
